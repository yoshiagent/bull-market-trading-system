Filter

濾網：大盤多頭

  • 大盤趨勢確認為多頭格局，才開始尋找個股機會
  • 空頭或盤整市場時，不操作、不進場
  • 判斷依據：指數結構是否呈現高點不斷墊高、低點不斷墊高的多頭排列
  • 可搭配均線（20MA / 60MA）確認多頭方向
✓ 多頭特徵
  • 高點逐步墊高
  • 低點逐步墊高
  • 均線多頭排列向上
✗ 暫停操作
  • 高點逐步降低
  • 低點逐步降低
  • 均線空頭排列向下
Entry Signal

進場訊號：N 形上漲

N 形結構示意
漲多 → 跌少（回檔） → 量增再漲突破
  • 順大勢逆小勢：大趨勢向上，利用小回檔低接
  • 只在關鍵價位進場：支撐區、前次低點、整數關卡
  • 多頭回檔量縮：下跌時成交量明顯縮減，顯示賣壓輕
  • 修正乖離：回測均線或修正過大乖離（MA20/MA60）
  • 最佳狀況：跌破關鍵價後假跌破，快速收回
  • 量增再漲：回檔結束，放量突破確認進場
Entry Checklist

進場確認清單

1
大盤確認多頭
濾網通過，大盤指數多頭格局成立
2
個股 N 形回檔
第一波拉升後，開始量縮回檔修正
3
抵達關鍵支撐 / 均線
回測前高、MA20/60、整數關卡或跌破假跌破
4
量增再漲，確認進場
出量向上，突破短期壓力，執行買入
口訣：大盤多 → 個股縮量回 → 到關鍵位 → 量增攻 → 進場
Operation

操作策略

🛑

做錯停損

進場後走勢不如預期，立即停損，不抱希望、不凹單

  • 跌破進場關鍵位
  • 訊號轉壞立刻出
📈

做對加碼

走勢確認，趁回檔加碼，讓獲利部位持續擴大

  • 每次回檔量縮再加
  • 金字塔加碼法
🌊

波段操作

不做短線當沖，持有至出場訊號，賺完整波段

  • 耐心持股波段走
  • 訊號未反轉不出
鐵律：輸家加碼輸更多，贏家加碼才是正確資金控管。做錯快砍，做對敢抱。
Exit Signal

出場訊號：反 N 形下跌

反 N 形結構示意
跌多 → 漲少（反彈） → 再跌破前低
  • 反 N 形結構確立：跌多漲少再跌多，型態成立
  • 跌破關鍵支撐：前低、均線全面失守
  • 反彈量縮無力，無法收復失地
  • 反彈至前高附近遇量縮壓回，是出場良機
  • 大盤濾網同步轉弱，全面出清
口訣：跌多 → 漲少 → 再跌多 → 前低不守 → 出場

Volume Spike Strategy

爆量操作策略

爆量 K 棒進場示意
等待收盤或即時價突破爆量K高點才進場
  • 低檔爆量（Volume Spike）：在相對低價區出現異常大量，代表大量換手、籌碼集中
  • 爆量當下不進場，先觀察等待
  • 當股價向上突破爆量 K 棒最高點時才進場
  • 邏輯：「大量的人都賺錢了」，持股成本整齊，後續上漲阻力小
  • 突破前若量縮整理，更是優質等待期
  • 停損設在爆量 K 棒最低點下方
口訣：低檔爆量 → 等整理 → 突破最高點 → 進場，籌碼乾淨漲勢順
Volume Spike Exit

爆量出場策略

高檔爆量 K 棒出場示意
等待收盤或即時價跌破爆量K最低點才出場
  • 高檔爆大量：在相對高價區出現異常大量，代表大量持股人解套或獲利了結出貨
  • 爆量當下不急出場，先觀察等待
  • 當股價向下跌破爆量 K 棒最低點時確認出場
  • 邏輯：「大量的人都賠錢了」，套牢籌碼沉重，每次反彈都是賣壓
  • 跌破後若反彈量縮無力，更確認出場訊號成立
  • 可搭配反 N 形型態一起判斷，雙重確認
口訣：高檔爆大量 → 等整理 → 跌破最低點 → 出場，套牢籌碼壓頭漲不動
